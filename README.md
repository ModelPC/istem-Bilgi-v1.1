# istem-Bilgi-v1.1

# 🖥️ Sistem Bilgi

Modern ve hafif bir macOS Sistem İzleme (System Monitor) uygulaması.  
Gerçek zamanlı CPU, RAM, Disk ve Network bilgilerini şık bir arayüzle gösterir.

---

## ✨ Özellikler

- 🧠 Gerçek zamanlı CPU kullanım takibi
- 📊 Animasyonlu CPU grafik (filled gradient)
- 🧩 Çekirdek bazlı CPU kullanım
- 🌡 CPU sıcaklık gösterimi
- 💾 RAM kullanım yüzdesi
- 💽 Disk kullanım oranı
- 🌐 Local & Public IP gösterimi
- 📈 Network trafik grafiği
- 🎨 Light / Dark / Auto tema desteği
- ⚙️ Ayarlar sekmesi (kalıcı AppStorage)
- ⏱ Yenileme süresi ayarlama (0.5 – 5 saniye)
- 🔘 İzlemeyi Başlat / Durdur
- ♻️ Grafik geçmişini sıfırlama
- 🍎 macOS 13+ uyumlu modern SwiftUI tasarım

---

## 🧰 Kullanılan Teknolojiler

- SwiftUI
- Combine
- AppStorage
- GeometryReader (Custom Charts)
- macOS 13 NavigationStack API

---

## 🖼️ Arayüz

Modern macOS Sonoma tarzı kart tasarım:
- Rounded glass effect
- Dynamic accent color
- Gradient chart rendering
- Responsive layout
